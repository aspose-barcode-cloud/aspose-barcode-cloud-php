# Code128Emulation

DEPRECATED. This enum will be removed in future releases Function codewords for Code 128 emulation. Applied for MicroPDF417 only. Ignored for PDF417 and MacroPDF417 barcodes.
## Allowable values

* **NONE**
* CODE903
* CODE904
* CODE905
