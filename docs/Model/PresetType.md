# PresetType

See QualitySettings allows to configure recognition quality and speed manually.
## Allowable values

* **HIGH_PERFORMANCE**
* NORMAL_QUALITY
* HIGH_QUALITY_DETECTION
* MAX_QUALITY_DETECTION
* HIGH_QUALITY
* MAX_BAR_CODES
